# Collaboration Skills

Skills for collaborative documentation and team alignment.

## Available Skills

### doc-coauthor
Structured workflow for co-authoring high-quality documentation.

**Capabilities**:
- Context gathering through structured questioning
- Section-by-section refinement
- Reader testing with fresh AI perspective
- Iterative improvement workflow

**When to use**:
- Writing PRDs that will be widely reviewed
- Creating technical specs for team alignment
- Drafting decision documents
- Any documentation requiring high quality

**Workflow**:
1. Context Gathering - Transfer all relevant context
2. Refinement & Structure - Build document section by section
3. Reader Testing - Verify document works for readers

**Documentation**: [doc-coauthor/SKILL.md](doc-coauthor/SKILL.md)

---

## Best Practices

1. Use for PRD writing in Phase 2
2. Use for architecture documentation in Phase 4
3. Invest time in context gathering - it pays off
4. Don't skip reader testing - it catches blind spots
